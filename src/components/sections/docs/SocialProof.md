# SocialProof Component

## Description
Section showcasing Naolito's massive social media presence and viral content creation capabilities. Features follower counts, engagement stats, and direct links to social platforms.

## Props
This component doesn't accept any props.

## Features
- Total followers and reach statistics
- Platform-specific follower counts (Instagram, TikTok, YouTube, Twitter)
- Viral content stats (views, videos, countries)
- Direct links to social media profiles
- Animated background gradients
- Responsive grid layout
- Dark theme optimized

## Usage Count
**Used 1 time**:
- Home page (between ClientLogos and FeaturedProjects)

## Dependencies
- None (fully self-contained)

## Example

```tsx
import SocialProof from '@/components/sections/SocialProof'

function HomePage() {
  return (
    <div>
      <Hero />
      <ClientLogos />
      <SocialProof />
      <FeaturedProjects />
    </div>
  )
}
```
