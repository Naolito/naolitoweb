# ServicesPreview Component

## Description
Minimalist, modern services section for the home page with numbered list items and hover interactions.

## Props
This component doesn't accept any props.

## Features
- Numbered service list with hover effects
- Dark background with dot pattern
- Arrow icons with animation on hover
- Links to full services page
- Mobile responsive

## Usage Count
**Used 1 time**:
- Home page (between FeaturedProjects and Footer)

## Dependencies
- react-router-dom (Link)

## Services Listed
1. Character Animation
2. Motion Graphics
3. 3D Animation
4. Viral Content

## Styling
- Background: Near-black (gray-950)
- Dot pattern overlay
- Border separators between items
- Hover: Text color change + arrow animation
