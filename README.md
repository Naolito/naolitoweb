# Naolito Animation Studio

Professional boutique animation studio website showcasing character-driven content for leading global brands including Netflix, Amazon, PlayStation, Disney, and Nestlé. Built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- **Professional Design**: Modern, clean aesthetic suitable for enterprise clients
- **Client Showcase**: Featuring Netflix, Amazon, PlayStation, Disney, Nestlé
- **5 Main Sections**: Home, Portfolio, About, Services, Contact
- **Boutique Positioning**: Premium, selective brand focus
- **Responsive Design**: Mobile-first approach
- **Future-Ready**: Prepared for edit mode integration with Supabase

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router** - Navigation

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/          # Reusable components
│   ├── layout/         # Header, Footer, Navigation
│   ├── ui/             # Buttons, Cards, Inputs, etc.
│   └── sections/       # Section-specific components
├── pages/              # Page components
├── store/              # State management
├── assets/             # Images, videos, fonts
├── types/              # TypeScript interfaces
└── styles/             # Global styles
```
